========
* Author  : [Hom](https://about.me/Hom)
* Version : v2.0
* Update  : 2016/01/12

Description
========
Camera-MT 是一個 Camera 模組，有彩色與黑白兩個版本，分別採用 MT9V034C12STC 和 MT9V034C12STM ，模組透過 8bits DCMI 與 SCCB 介面作暫存器的讀寫和影像的讀取。
